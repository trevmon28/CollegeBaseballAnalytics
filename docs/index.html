How the Power Score Works

Each team gets a composite score built from six metrics, all Z-score normalized so conferences with different run environments are compared fairly. Weights:

  • Pythagorean W% (25%) — expected win rate from runs scored vs allowed
  • Run Differential / G (20%) — average margin per game
  • Elo Rating (20%) — opponent-adjusted performance over all seasons
  • Runs Scored / G (10%) — offensive output
  • Runs Allowed / G (10%) — pitching / defense (lower is better)
  • SOS — Avg Opp Elo (15%) — strength of schedule

Elo Rating System

Elo is a running skill rating updated after every game. Key adjustments:

  • Home advantage: +35 Elo points added to the home team before each game
  • Margin multiplier: log(|run diff| + 1) x 2 — bigger wins move the needle more
  • Conference tier K-factor: games between stronger conferences generate more Elo movement (SEC/ACC/Big 12 = 1.0x, down to Ohio Valley = 0.65x)
  • Starting value: 1500 for all teams before their first game

Conference Tiers

Tiers scale the K-factor so low-major games generate less Elo movement per win:

SEC 1.00 ACC 1.00 Big 12 1.00 Sun Belt 0.90 AAC 0.90 Big Ten 0.90 C-USA 0.78 WAC 0.78 OVC 0.65

Top-25 Eligibility (P-Conf G)

To appear in the Top 25 chart and headline ranking, a team must meet one of:

  • Play in a power conference (tier ≥ 0.90) — auto-eligible
  • OR have played at least 12 games against power-conference opponents in the current season

This prevents mid-major teams that only beat cupcakes from inflating into the top 25. The P-Conf G column in the table shows each team's count. Ineligible rows are dimmed.

Strength of Schedule (SOS)

SOS is the average Elo rating of all opponents faced in the current season. A higher number means a harder schedule. It accounts for 15% of the composite power score, so teams that only play weak opponents are penalized even if their win-loss record looks strong.

Recent Form (L15)

Rolling win percentage over the last 15 games (minimum 3 games). Shown in Team Profile to highlight teams on a hot or cold streak that the season-aggregate stats may not yet reflect. Not included in the composite power score — use it as a qualitative check.

Pythagorean Win %

Expected win percentage based on runs scored vs runs allowed, using the formula popularized by Bill James: RS^2 / (RS^2 + RA^2). It strips out luck in close games and is a better predictor of future performance than actual win percentage.

Metric Quick Reference

  • Elo — opponent-adjusted skill rating (higher = better)
  • SOS — avg Elo of opponents faced (higher = harder schedule)
  • Rd/G — run differential per game (+ is good)
  • Pythag W% — luck-adjusted win rate
  • RS/G — runs scored per game (offense)
  • RA/G — runs allowed per game (pitching/defense)
  • P-Conf G — games played vs power-conf opponents
  • Score — composite power score (Z-score weighted sum)
